# Tailwind-React-UI-TS

Typescripted UI Library for React using Tailwind. Built off the work from [tailwind-react-ui](https://github.com/emortlock/tailwind-react-ui)

## Usage

To install dependencies
```
$ yarn
```

To build
```
$ yarn build
```

To run tests
```
$ yarn test
```

To run Storybook
```
$ yarn storybook
```
